### 리팩토링
- main에 로직을 service로 분리
  - main은 객체를 종합해서 실행하는 진입점의 역할만 하도록 수정
  - 여러 역할의 로직이 한번에 포함되서 이해하기 어려운 코드라고 생각해서 리팩토링
